## Education & PROcrastination Language

A very simple, minimal, low-level WASM language.
Think of it as a bare-bones ALGOL-like syntactical abstraction over WAT with
standardised strings, arrays and records.

This language is still in very early development and does NOTHING, come back in
like half a year and there might be something to to interest you.
